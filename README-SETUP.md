# TRW.Rise Website Setup Guide

This folder contains the finished TRW.Rise homepage files.

## Files included

- `index.html` — the page structure and website copy
- `styles.css` — the full design, colors, spacing, mobile layout, and branding
- `script.js` — mobile menu and signup button behavior
- `assets/trw-rise-official-logo.jpeg` — official website logo
- `assets/trw-rise-mark.jpeg` — secondary brand mark and visual watermark asset

## Best Replit setup

1. Open your Replit project.
2. In the file panel, find your current website files.
3. Replace the current `index.html` with the new `index.html`.
4. Replace the current `styles.css` with the new `styles.css`.
5. Replace the current `script.js` with the new `script.js`.
6. Create an `assets` folder if one does not already exist.
7. Upload both logo files into the `assets` folder.
8. Click Run.
9. Open the site preview.

## Important

Keep the file names exactly the same unless you also update the code.

These paths are already built into the site:

- `assets/trw-rise-official-logo.jpeg`
- `assets/trw-rise-mark.jpeg`

If the logo does not show, the asset folder or file name is wrong.
